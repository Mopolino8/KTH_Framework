Detailed Description

Set of multi-file checkpoint routines for DNS and linear.

VERSION FOR DNS AND LINEAR SIMULATIONS This version supports only proper restart for DNS and for perturbation mode with single(!!!!!) NPERT=1 perturbation and not advected basefield IFBASE=F. For DNS only 'rs8' files are written. For perturbation mode files 'rs8' include perturbation data and the basefield is saved in 'rb8'.
