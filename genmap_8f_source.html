3 c VERSION WITH SIMPLE BISECTION AND GROWTH OF CONN SET, COMBO (klh 07/09)
4 c
5 c genmap.f performs the following operations:
6 c
7 c 1) Read a nek5k .rea or (.rea/.re2) file describing problem geometry - 2D or 3D.
8 c
9 c 2) Construct a standard finite element mesh from this file, i.e.,
10 c
11 c a. a list of coordinates, X1, X2,...,X_npts, where X_i = (xi,yi,zi) in 3D
12 c or X_i = (xi,yi) in 2D
13 c
14 c
15 c b. a list of E cells, each cell comprising 2^d integers, d=2 or 3, where
16 c E is the number of spectral elements specified in the .rea file.
17 c Each integer in a cell points to one of the coordinate X_i above.
18 c
19 c 3) Generate the graph Laplacian, G, for the _dual_ graph corresponding to the
20 c FE mesh. Each node in the dual graph is an element. Each edge in the
21 c graph is an element G_ij, for any element i connected to j.
22 c
23 c . G_ij = 1 if element i shares a vertex with element j
24 c . G_ij = 2 if element i shares an edge with element j
25 c . G_ij = 4 if element i shares an face with element j
26 c
27 c . G_ii = - \sum j^=i G_ij
28 c
29 c
30 c 4) Partition this graph using the Fiedler vector generated by a restarted
31 c Lanczos iteration.
32 c
33 c 5) Recur on (3)-(4) in a depth-first traversal of the subgraphs until
34 c P branches have been identified, where P = floor(log_2 E).
35 c
36 c
37 c 6) Output the results
38 c
39 c-----------------------------------------------------------------------
40 c
41 c This code no longer relies on Metis to do the partitioning.
42 c
43 c For large problems ( nel > 1e6 ), compile with -mcmodel=medium and
44 c change parameter lelm (maximum number of elements)
45 c
46 c
47 c genmap() uses the symmetric vertex ordering
48 c
49 c 3 ----- 4
50 c | |
51 c | |
52 c | |
53 c 1 ----- 2
54 c
55 c which can be extended to an arbitrary number of space dimensions
56 c (like 3).
57 c
58 c
59 c
60 c 7/27/07 -- add self-connected check
61 c 7/27/07 -- verify that don't double-check periodic faces
62 c
63 c 1/13/09 -- added support for conjugate heat transfer;

66 c .recursive bisection only on fluid graph
