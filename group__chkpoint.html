Detailed Description

Multi-file checkopinting for KTH submodules.

Module interface:
This module provides following interface:
  1. checkpoint_main
This module requires following interfaces:
  1. chkpt_init
  2. chkpt_read
  3. chkpt_write
Module interface usage:
subroutine userchk
! save/load files for full-restart
call checkpoint_main
return
end
Module parameters:
Global parameter list:
Parameters provided by checkpoint module (include file - CHKPOINTD):
  • IFCHKPTRST - (logical) if restart
  • CHKPTSTEP - (integer) checkpiont dump frequency (number of time steps)
Module parameter usage:
&chkpoint
chkptstep= 10,
ifchkptrst=f,
