Detailed Description

Multi-file checkopinting for DNS and linear simulations.

In the case of multistep time-integration method one needs data from NBDINP timestep. I use standard full_restart and full_restart_save subroutines. There are four 'rs8...' reatart files saved in double precission. Only .f format is supported. In any case two sets of restart files (1-4;5-8) are created.

Attention
To make this varsion to work correctly mod(total_number_of_steps,CHKPTSTEP).ge.NBDINP Otherwise last checkpoint will not be complete.
Module interface:
This module provides following interfaces to checkpoint:
  1. chkpt_init
  2. chkpt_read
  3. chkpt_write
Module parameters:
Global module dependency:
Parameters required by dns_lin module:
  1. CHKPOINTD:
    • IFCHKPTRST - if restart
    • CHKPTSTEP - checkpiont dump frequency (number of time steps)
  2. Nek5000 PARAM array
    • PARAM(66) - write format
    • PARAM(67) - read format
