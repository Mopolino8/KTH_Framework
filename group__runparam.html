Detailed Description

User/module runtime parameters.

Deprecated:
This module and its submodules are deprecated. It is currently replaced by Nek5000 dictionary.

Set of subroutines to read user and module parameters using Fortran90 namelists.

Module interface:
This module provides following interface:
  1. runprm_main
This module requires following interfaces:
  1. user_param_in(iunit)
  2. user_param_out(iunit)
Module interface usage:
subroutine usrdat
! to read user/module runtime parameters
call runprm_main
return
end
Attention
runprm_main has to be executed before any of module initialisation routine is called.

subroutine runprm_main ( )

Main interface of runtime parameter module.

This routine reads runtime parameters from .rpar file using Fortran90 namelists and writes them to standard outpu (log file).

Attention
This routine should be executed before any of module initialisation routnes would be called (e.g. inside usrdat).
